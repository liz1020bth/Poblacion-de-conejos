01 · Contexto del Problema

¿Cómo crecen los conejos
con el tiempo?

En el año 1202, el matemático italiano Leonardo Fibonacci planteó este enigma: si ponemos una pareja de conejos en un campo cerrado, ¿cuántas parejas habrá después de N meses? La respuesta dio origen a una de las secuencias matemáticas más famosas de la historia.

📜

El problema original de Fibonacci

Leonardo de Pisa (apodado Fibonacci) supuso que: un par de conejos tarda un mes en madurar, luego produce un nuevo par cada mes, y los conejos nunca mueren. Bajo estas reglas, la población sigue exactamente la serie: 1, 1, 2, 3, 5, 8, 13…

🌿

¿Por qué importa esto hoy?

Este mismo patrón aparece en la biología de poblaciones, el crecimiento bacteriano, la disposición de pétalos en flores, y los modelos económicos de crecimiento compuesto. Entender Fibonacci es entender cómo crece la naturaleza.

🔢

La regla matemática

Cada número de la serie es la suma de los dos anteriores:

F(1) = 1  |  F(2) = 1
F(n) = F(n−1) + F(n−2)

Simple, pero produce un crecimiento exponencial asombroso.

📊

Crecimiento real de la colonia

A los 10 meses: 55 pares (110 conejos).
A los 20 meses: 6,765 pares.
A los 30 meses: 832,040 pares.

El crecimiento de Fibonacci aproxima el crecimiento exponencial real de muchas especies.

02 · Explicación del Algoritmo

¿Cómo funciona el algoritmo?

El algoritmo calcula la serie de Fibonacci usando solo tres variables (sin arreglos ni arrays), tal como lo establece el desafío. Así funciona paso a paso:

🌱 Paso a paso del algoritmo

Empezamos con a = 1 y b = 1 (los dos primeros meses). En cada iteración calculamos c = a + b, luego avanzamos: a = b, b = c. Sin usar ningún array — solo tres variables simples.

// Variables simples (sin arrays) let a = 1, b = 1, c; for (let i = 3; i <= meses; i++) { c = a + b; // nuevo término a = b; // desplazar b = c; // siguiente }
